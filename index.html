Python

Python is a very popular language that offers career flexibility and a good basis for understanding the logic behind programming. It is used by the likes of Facebook and Google, and many other big companies. Job opportunity galore! It also has one of the highest average salaries behind Ruby and Matlab! But remember, most languages are lucrative and desirable skills so don’t let that factor too much into your final choice! It is great for creating prototypes quickly, but is also used for more robust ventures like machine learning and data science.

Some things built with Python:

Instagram - Spotify - Dropbox - Reddit


What is it used for?

  • Web Development
  • Video Game Development
  • Desktop Graphic User Interfaces
  • Software Development
  • Machine Learning and Artificial Intellignece

Pros of Python

  1. Easy to learn
  2. Simple concise syntax
  3. A lot of documentation and a big community for support
  4. Extensive libraries, tools, and functionality
  5. Use in the Internet of Things
  6. Ability ot scale projects
  7. Support of multiple platforms and systems

Cons of Python

  1. Slower than compiled languages
  2. Low memory management
  3. Poor mobile computing-not used in many smartphone apps
  4. Language design requires more testing and has more errors
  5. Starting with its simple syntax can make future languages harder

JavaScript

Great for aspiring web developers. You seem like a great fit for front end design with your strong intuition and communication skills. And while jQuery and Angular frameworks can help with client-side endeavors, NodeJS can also be used for any back-end programming. This is one of the most sought after languages across the United States, so you could work almost anywhere! You'll be able to create dynamic content, control multimedia, and be able to create everything you see on other websites. We suggest you learn this one alongside HTML and CSS since JavaScript is commonly embedded into HTML code. If interactivity is your name, JavaScript is your game!


Pros of JavaScript

  1. A great StackOverflow community ready to answer your questions
  2. Relative simplicity makes it more accessible for new programmers
  3. Versatility with the ability to be used with many other languages and for different applications
  4. Speed thanks to client-side nature

Cons of JavaScript

  1. Security due to user execution that may be exploited
  2. Reliance on user causes unpredictable client-side scripts (be sure to test!)
  3. Interpreted differently by different browsers
  4. Single errors can stop the entire JavaScript code from rendering

What is it used for?

  • Front End of Websites
  • Analytics
  • Widgets
  • Web Interactivity

Some things built with JavaScript:

PayPal - YouTube - Wikipedia - LinkedIn


C#

C# is powerful and flexible, and able to make many different applications only limited by your own imagination. With no constraints you can make anything from websites, developer tools, to compilers. It is just behind Java when it comes to employer demand, so you'll be a highly sought-after candidate. Your creative energy is sure to shine if you don't mind a bit of a challenge! It is also a great language for anyone hoping to work at Microsoft. Your emphasis on simplicity and efficiency makes you and C# a perfect match!

Some things built with C#:

StackOverflow - Evernote - Microsoft Visual Studio


What is it used for?

  • Windows Applications
  • Business Enterprise Applications
  • Software Development
  • Video Games

Pros of C#

  1. Statistically-typed so errors are detected before applications go live
  2. Scalable and easy to maintain
  3. A large, helpful community
  4. Windows .NET framework integration for great functionality and support
  5. Transferable language that offers a springboard to Java, Objective C, PHP, and C++
  6. High demand

Cons of C#

  1. Steeper learning curve for newcomers
  2. Loss of cross-platform capabilities due to .NET integration
  3. Proprietary, upfront cost to purchase software
  4. Runs slowly and needs to be compiled each time changes are made
